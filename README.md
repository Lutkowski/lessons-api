# Lessons-api
HTTP сервис на базе ExpressJS, выполняющий поиск занятий на базе фильтров

# Запуск проекта
```bash
npm install
```
создать .env файл по образцу из .env.example
```bash
npm run start
```

# Поиск работает в два этапа
-Применение фильтров к lessons и нужным подзапросам с последующей пагинацией. Результат - список id lessons, соответсвующий фильтрам и учитывающий пагинацию.
-Получение полной информации. Left join таблиц, для выбранных на первом этапе id.
